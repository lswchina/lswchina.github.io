# Suwan Li - Personal Homepage

This repository contains the personal homepage of Suwan Li, a Ph.D. student in Computer Science and Technology at Nanjing University.

## Contents

- `index.html` - Main HTML file
- `style.css` - Custom CSS styles
- `README.md` - This file

## Features

- Responsive design using Bootstrap 5
- Clean, modern interface with custom CSS
- Sections: About, Awards, Publications, Service, Contact
- Smooth scrolling navigation
- Mobile-friendly layout

## How to Deploy on GitHub Pages

1. Create a new repository on GitHub (e.g., `suwanli.github.io` or `suwanli-homepage`)
2. Push the contents of this folder to the repository
3. Go to repository Settings > Pages
4. Under "Source", select "Deploy from a branch"
5. Choose the branch (e.g., `main`) and folder (`/root`)
6. Save and wait for deployment (usually takes a minute)
7. Your site will be available at `https://[your-username].github.io/[repository-name]`

If you want to use a custom domain, add a `CNAME` file with your domain name.

## Local Development

To view the site locally, simply open `index.html` in a web browser.

## Customization

1. **Profile Photo**: Replace the placeholder image in the `hero-section` with your own photo
2. **Contact Information**: Update email, phone, and address in the HTML
3. **Social Links**: Replace `#` in social media links with your actual profile URLs
4. **Content**: Modify the text in each section as needed
5. **Colors**: Edit the CSS variables in `style.css` to change the color scheme
